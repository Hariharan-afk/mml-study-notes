# Mathematics for Machine Learning — Study Notes

Welcome to my study notes for **Mathematics for Machine Learning**.

These notes are designed as a companion resource for learners who want intuitive explanations, simple examples, visual interpretations, and machine learning connections.

## What these notes focus on

These notes emphasize:

- intuition before formalism,
- small numerical examples,
- geometric meaning,
- machine learning interpretation,
- common confusions,
- Python/NumPy demos where useful.

## How to read these notes

If you are new to the mathematics behind machine learning, start with:

1. Linear Algebra
2. Analytic Geometry
3. Matrix Decompositions
4. Vector Calculus
5. Probability and Distributions
6. Continuous Optimization

After that, move to the machine learning chapters:

1. Linear Regression
2. Principal Component Analysis
3. Gaussian Mixture Models
4. Support Vector Machines
