# Chapter 2: Linear Algebra

Linear algebra studies vectors, matrices, and the rules for manipulating them.

In machine learning, linear algebra is used to represent:

- data points as vectors,
- datasets as matrices,
- model parameters as vectors,
- transformations as matrices,
- systems of equations as matrix equations.

## Big picture

Linear algebra gives us the language to describe data, models, and transformations.

The central idea is that many machine learning problems can be written using vectors and matrices.
