# Learning Roadmap

This roadmap follows the structure of *Mathematics for Machine Learning*.

## Part I: Mathematical Foundations

| Chapter | Topic | Why it matters |
|---|---|---|
| 2 | Linear Algebra | Represents data, transformations, systems of equations, and model parameters |
| 3 | Analytic Geometry | Defines distance, similarity, angles, orthogonality, and projections |
| 4 | Matrix Decompositions | Reveals hidden structure in matrices and data |
| 5 | Vector Calculus | Provides gradients for learning and optimization |
| 6 | Probability and Distributions | Models uncertainty and noisy data |
| 7 | Continuous Optimization | Explains how models learn parameters |

## Part II: Machine Learning Problems

| Chapter | Topic | Main idea |
|---|---|---|
| 8 | Models and Data | Connects data, models, and learning |
| 9 | Linear Regression | Fits linear models to data |
| 10 | PCA | Finds low-dimensional structure in data |
| 11 | Gaussian Mixture Models | Models data as a mixture of probability distributions |
| 12 | Support Vector Machines | Learns separating boundaries for classification |
