# Copyright and Use

These notes are created as a companion learning resource for *Mathematics for Machine Learning* by Marc Peter Deisenroth, A. Aldo Faisal, and Cheng Soon Ong.

## Purpose of These Notes

The purpose of this project is to help learners understand the mathematical foundations of machine learning through:

- plain-English explanations,
- original examples,
- visual intuition,
- Python/NumPy demos,
- machine learning connections,
- common confusions and clarifications.
