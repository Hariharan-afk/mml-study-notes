# Note Writing System

This page defines the writing workflow for this project.

The goal is not to summarize the book line by line. The goal is to create a beginner-friendly companion resource that explains the ideas clearly using intuition, examples, diagrams, and machine learning connections.

## Core Workflow

For every concept, follow this loop:

```text
Read → Understand → Solve → Explain → Visualize → Code → Publish → Revise
```

## Three-Pass Method

### Pass 1: Rough Understanding

Write private rough notes in the scratchpad/ folder.

Focus on:

- what the section is trying to teach,
- new symbols,
- confusing parts,
- important examples,
- possible machine learning connections.

### Pass 2: Active Recall

Close the book and explain the concept in your own words.

Use questions like:

- What problem does this concept solve?
- What is the simplest example?
- What does it mean geometrically?
- Where does this appear in machine learning?
- What would confuse a beginner?

### Pass 3: Public Note

Convert the rough explanation into a clean public note under docs/.

Every public note should include:

- Why this topic matters
- Intuition
- Formal definition
- Simple example
- Geometric interpretation
- Machine learning connection
- Common confusion
- Python/NumPy demo when useful
- Summary
- References

### Quality Levels

#### Level 1: Basic

- Clear title
- Simple motivation
- Correct definition
- Small example

#### Level 2: Useful

- Intuition
- Geometric meaning
- ML connection
- Common confusion
- Summary

#### Level 3: Strong

- Original diagram
- Python/NumPy demo
- Practice question
- Cross-links to related notes

#### Level 4: Excellent

- Step-by-step derivation
- Multiple interpretations
- Visual explanation
- Connection to later ML chapters
- Reviewed after finishing related chapters

Aim for Level 2 before publishing. Upgrade to Level 3 or 4 during revision.

## Copyright-Safe Rule

Do not copy paragraphs, figures, exercises, or solutions from the book.

Write explanations in your own words.

It is okay to reference chapter and section numbers.
