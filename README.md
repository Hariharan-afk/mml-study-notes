# Mathematics for Machine Learning — Study Notes

Beginner-friendly study notes, explanations, examples, diagrams, and Python demos created while studying **Mathematics for Machine Learning** by Marc Peter Deisenroth, A. Aldo Faisal, and Cheng Soon Ong.

The goal of this repository is to build a clear companion resource for learners who want to understand the mathematical foundations behind machine learning.

## Purpose

These notes are written to help learners understand:

- why each mathematical concept matters,
- how to interpret equations intuitively,
- how concepts connect to machine learning,
- how to work through small examples,
- how to implement ideas using Python and NumPy.

## Important Disclaimer

These are my personal study notes written in my own words.

This repository is **not** a replacement for the official book and does not reproduce the book’s content, figures, exercises, or solutions.

Please refer to the official book website:

https://mml-book.com

## Learning Philosophy

Each note follows this structure:

1. Motivation
2. Intuition
3. Formal definition
4. Simple example
5. Geometric interpretation
6. Machine learning connection
7. Common confusion
8. Python/NumPy demo when useful
9. Summary

The aim is not just to summarize the book, but to explain the ideas in a way that is easier to revisit and teach.
