# Contributing

Thank you for your interest in contributing to this project.

This repository contains beginner-friendly study notes for *Mathematics for Machine Learning*.

## Contribution Goals

Good contributions should improve:

- clarity,
- intuition,
- correctness,
- examples,
- diagrams,
- Python/NumPy demos,
- machine learning connections,
- explanations of common confusions.

## Copyright Rules

Please do not contribute:

- copied text from the book,
- screenshots of book pages,
- copied figures from the book,
- copied exercises,
- official solutions,
- redistributed copies of the book PDF.

Contributions should be written in your own words.

It is okay to reference chapter and section numbers from the book.

## Note Style

Each concept note should generally follow this structure:

1. Why this topic matters
2. Intuition
3. Formal definition
4. Simple example
5. Geometric interpretation
6. Machine learning connection
7. Common confusion
8. Python/NumPy demo when useful
9. Summary
10. References

## Pull Requests

Before opening a pull request, please make sure:

- the explanation is written in your own words,
- equations render correctly,
- examples are checked,
- code snippets run successfully if included,
- the note is beginner-friendly.

## Local Development

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the site locally:

```bash
mkdocs serve
```

Build the site:

```bash
mkdocs build
```
